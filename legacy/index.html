johnnydmad Legacy

These tracks are carried over from johnnydmad's predecessors, Beyond Chaos EX and nascentorder. They use a combination of Final Fantasy VI vanilla samples and a custom sample set originally based on FF6 T-Edition's. The long-term goal is to eventually replace all of these with updated versions using the newer johnnydmad sample set.
